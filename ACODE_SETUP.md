# West Bengal Monitor - Acode Setup Guide 📱

## Quick Start for Acode Mobile Editor

### Step 1: Install Acode
- Download **Acode** from Google Play Store
- Open the app on your Android device

### Step 2: Clone Repository in Acode
1. Open Acode
2. Tap **"+"** to create new project
3. Select **"Clone from Git"**
4. Paste: `https://github.com/Education1234-dot/west-bengal-monitor-ai.git`
5. Wait for download to complete

### Step 3: Project Structure in Acode
```
west-bengal-monitor-ai/
├── mobile-app/          ← MOBILE VERSION (Recommended)
│   ├── index.html
│   ├── style.css
│   ├── app.js
│   └── manifest.json
├── backend/
├── frontend/
├── docker-compose.yml
└── README.md
```

### Step 4: Open Files in Acode
**For Mobile App (No Dependencies)**
- Open: `mobile-app/index.html` - View structure
- Open: `mobile-app/style.css` - Edit styles
- Open: `mobile-app/app.js` - Edit JavaScript

**For React/Node.js**
- Install Node.js on your device (via Termux if needed)
- Edit `frontend/src/App.js` in Acode
- Edit `backend/app.py` in Acode

### Step 5: Testing
- **HTML/CSS/JS Preview**: Long press file → Preview
- **Backend**: Use Termux for Python
- **Frontend**: Use Termux for npm start

## File Navigation in Acode
- 📁 Folder icon: Browse files
- 🔍 Search icon: Find code
- ⚙️ Settings: Configure editor

## Keyboard Shortcuts (Acode)
- `Ctrl+S` - Save file
- `Ctrl+F` - Find in file
- `Ctrl+H` - Find & Replace
- `Ctrl+/` - Comment code
- `Tab` - Indent code
